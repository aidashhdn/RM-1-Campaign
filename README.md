# RM-1-Campaign
A website for RM1 Campaign 

For javaScript I put 3 scrolling buttons,
-go to header button
-go to main button
-go to footer button

when clicking these buttons, it will trigger the 'scrollToSection" function with the corresponding
ID of the section(header,main,footer)
